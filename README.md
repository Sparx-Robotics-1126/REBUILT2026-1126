# REBUILT2026-1126

Robot code for FIRST Robotics "REBUILT" 2026 season for [FRC Team 1126 - SparX](https://gosparx.org). This code base is based on the [GRRBase](https://github.com/Greater-Rochester-Robotics/GRRBase) created by [FRC Team 340 - G.R.R.](https://team340.org/).

## Highlights

### From GRRBase
- [Swerve API](src/main/java/org/team340/lib/swerve/SwerveAPI.java)
- [Pre-defined Loggers](src/main/java/org/team340/lib/logging) 
- [Various other utilities](src/main/java/org/team340/lib/util)

### Other Features
- [Photo Vision](https://photonvision.org/)

Used for April Tag naivation and pose. 

## Getting Started
### Prerequisites

- [WPILib](https://github.com/wpilibsuite/allwpilib/releases) 2026.x
- [Node.js](https://nodejs.org/en/download) (v24 LTS Recommended)

Node.js is required to support linting via [Spotless](https://github.com/diffplug/spotless), using the [Prettier](https://prettier.io) style guide. You can apply these rules by building, or by running `./gradlew spotlessApply`.
